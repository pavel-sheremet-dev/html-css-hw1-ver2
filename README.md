# Моменти, які потрібно врахувати в ТЗ

1. Хедер, футер і секції повинні мати `id`:
- <header id="header">
- <section id="hero">
- <section id="features">
- <section id="team">
- <section id="portfolio"> (починаючи з 2 ДЗ)


2. Посилання на логотип потрібно робити окремо від `nav` (`nav` буде включати навігацію по самій сторінці, логотип просто веде на індексну сторінку)

3. Навігаційні посилання - це якірні посилання:

```html
<a href="#features">Features</a>
<a href="#team">Team</a>
<a href="#portfolio">Portfolio</a>
```

Відповідні зміни потрібно зробити в макеті.

4. Для секції переваг використовувати заголовок "Our features" (з поясненням, що починаючи з 3 модуля він буде коректно прихований).
